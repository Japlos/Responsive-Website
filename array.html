1. JavaScript Arrays

In JavaScript, an array is a special type of object used to store a list of values. Arrays are zero-indexed, meaning the first element has an index of 0. Arrays can store elements of any type: numbers, strings, objects, or even other arrays.

1.1 Declaring an Array

Arrays can be declared using square brackets ([]) or the Array constructor. The most common way to declare an array is using the square brackets syntax.

let fruits = ["Apple", "Banana", "Orange"];
let numbers = new Array(1, 2, 3, 4);

1.2 Accessing Array Elements

Array elements can be accessed using their index. The index starts from 0 for the first element.

let fruits = ["Apple", "Banana", "Orange"];
console.log(fruits[0]); // Outputs: Apple

1.3 Array Length

The length property returns the number of elements in an array.

let fruits = ["Apple", "Banana", "Orange"];
console.log(fruits.length); // Outputs: 3

2. JavaScript Array Methods

JavaScript arrays come with built-in methods that allow for manipulation and transformation of array data. Here are some commonly used array methods:

2.1 push()

The push() method adds one or more elements to the end of an array and returns the new length of the array.

let fruits = ["Apple", "Banana"];
fruits.push("Orange");
console.log(fruits); // Outputs: ["Apple", "Banana", "Orange"]

2.2 pop()

The pop() method removes the last element from an array and returns it.

let fruits = ["Apple", "Banana", "Orange"];
let lastFruit = fruits.pop();
console.log(lastFruit); // Outputs: Orange
console.log(fruits); // Outputs: ["Apple", "Banana"]

2.3 shift()

The shift() method removes the first element of an array and returns it.

let fruits = ["Apple", "Banana", "Orange"];
let firstFruit = fruits.shift();
console.log(firstFruit); // Outputs: Apple
console.log(fruits); // Outputs: ["Banana", "Orange"]

2.4 unshift()

The unshift() method adds one or more elements to the beginning of an array and returns the new length of the array.

let fruits = ["Banana", "Orange"];
fruits.unshift("Apple");
console.log(fruits); // Outputs: ["Apple", "Banana", "Orange"]

2.5 splice()

The splice() method changes the contents of an array by removing or replacing existing elements and/or adding new elements.

let fruits = ["Apple", "Banana", "Orange"];
fruits.splice(1, 1, "Mango", "Pineapple");
console.log(fruits); // Outputs: ["Apple", "Mango", "Pineapple", "Orange"]

2.6 slice()

The slice() method returns a shallow copy of a portion of an array into a new array object.

let fruits = ["Apple", "Banana", "Orange", "Grapes"];
let selectedFruits = fruits.slice(1, 3);
console.log(selectedFruits); // Outputs: ["Banana", "Orange"]

2.7 concat()

The concat() method combines two or more arrays and returns a new array without modifying the original arrays.

let fruits = ["Apple", "Banana"];
let moreFruits = ["Orange", "Grapes"];
let allFruits = fruits.concat(moreFruits);
console.log(allFruits); // Outputs: ["Apple", "Banana", "Orange", "Grapes"]

3. JavaScript Array Search

JavaScript provides several methods for searching through arrays. These methods help find specific elements or check for their existence in the array.

3.1 indexOf()

The indexOf() method returns the index of the first occurrence of a specified element, or -1 if it is not found.

let fruits = ["Apple", "Banana", "Orange"];
console.log(fruits.indexOf("Banana")); // Outputs: 1
console.log(fruits.indexOf("Grapes")); // Outputs: -1

3.2 includes()

The includes() method checks if an array contains a specific element and returns true or false.

let fruits = ["Apple", "Banana", "Orange"];
console.log(fruits.includes("Banana")); // Outputs: true
console.log(fruits.includes("Grapes")); // Outputs: false

3.3 find()

The find() method returns the first element in the array that satisfies the provided testing function.

let numbers = [4, 9, 16, 25];
let found = numbers.find(x => x > 10);
console.log(found); // Outputs: 16

4. JavaScript Array Sort

JavaScript provides the sort() method to sort the elements of an array. The array is sorted in place, meaning the original array is modified.

4.1 Sorting Strings

The sort() method can be used to sort strings in alphabetical order by default.

let fruits = ["Banana", "Apple", "Orange"];
fruits.sort();
console.log(fruits); // Outputs: ["Apple", "Banana", "Orange"]

4.2 Sorting Numbers

When sorting numbers, you need to provide a compare function because the sort() method sorts numbers as strings by default.

let numbers = [4, 9, 1, 5, 3];
numbers.sort((a, b) => a - b);
console.log(numbers); // Outputs: [1, 3, 4, 5, 9]

5. JavaScript Array Iteration

JavaScript arrays come with a variety of methods for iterating over elements. These methods allow you to perform operations on each element of the array.

5.1 forEach()

The forEach() method executes a provided function once for each array element.

let fruits = ["Apple", "Banana", "Orange"];
fruits.forEach((fruit) => {
    console.log(fruit);
});
// Outputs:
// Apple
// Banana
// Orange

5.2 map()

The map() method creates a new array populated with the results of calling a provided function on every element in the array.

let numbers = [1, 2, 3, 4];
let doubledNumbers = numbers.map(x => x * 2);
console.log(doubledNumbers); // Outputs: [2, 4, 6, 8]

6. JavaScript Array Const

When an array is declared using the const keyword, the reference to the array cannot be reassigned, but the array's elements can still be modified.

6.1 Example of const with Arrays

You cannot reassign a new array to a const array, but you can change the array's elements.

const fruits = ["Apple", "Banana"];
fruits.push("Orange"); // Allowed
console.log(fruits); // Outputs: ["Apple", "Banana", "Orange"]

// Reassigning the array will result in an error:
fruits = ["Grapes"]; // Error: Assignment to constant variable
