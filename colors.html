HTML Colors

In HTML, colors can be specified by name, hex code, RGB, RGBA, HSL, or HSLA values. Colors help make a webpage visually appealing and easier to navigate.

HTML CSS

CSS, or Cascading Style Sheets, is used to style HTML elements. CSS can be written inline, internally within the HTML file, or externally in a separate CSS file.

Inline CSS

This text has inline CSS applied.

Internal CSS

Internal CSS is written in the <style> tag within the <head> section of the HTML document.

        <style>
          p { color: green; font-weight: bold; }
        </style>

External CSS

External CSS is written in a separate file (e.g., style.css) and linked to the HTML file.

        <link rel="stylesheet" href="style.css">

HTML Favicon

A favicon is a small icon that appears in the browser tab next to the title of the webpage. Favicons are typically 16x16 or 32x32 pixels and are saved as .ico, .png, or .svg files.

How to Add a Favicon

To add a favicon, use the <link> tag in the <head> section:

        <link rel="icon" href="favicon.ico" type="image/x-icon">

You can create favicons using online tools or image editors that support .ico files.
