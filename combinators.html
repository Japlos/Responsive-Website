1. CSS Combinators

CSS combinators are used to define relationships between selectors. They allow you to apply styles based on the position of elements relative to other elements. There are four primary combinators: descendant, child, adjacent sibling, and general sibling.

Types of Combinators:

  • Descendant Combinator (space): The descendant combinator applies styles to any element that is a descendant (child, grandchild, etc.) of another element. The space between two selectors indicates a descendant relationship.
    • div p {
    color: blue;
}

    This applies the color blue to all <p> elements inside a <div>, including nested paragraphs.

  • Child Combinator (>): The child combinator selects elements that are a direct child of a specific element. The child relationship is represented by the > symbol.
    • div > p {
    color: red;
}

    This applies the color red to only the <p> elements that are direct children of a <div> element, not nested paragraphs.

  • Adjacent Sibling Combinator (+): The adjacent sibling combinator selects an element that is immediately adjacent to a specified element. It uses the + symbol to indicate adjacency.
    • h1 + p {
    font-size: 18px;
}

    This applies a font size of 18px to the <p> element that directly follows an <h1> element.

  • General Sibling Combinator (~): The general sibling combinator applies styles to all elements that are siblings (i.e., share the same parent) and appear after a specified element. It uses the ~ symbol to indicate a sibling relationship.
    • h1 ~ p {
    color: green;
}

    This applies the color green to all <p> elements that are siblings of an <h1> element and come after it in the HTML document.

Example of Combinator Usage:

ul > li {
    font-weight: bold;
}

div p {
    font-style: italic;
}

This example makes the direct child <li> elements inside a <ul> bold and the <p> elements inside a <div> italic.

2. CSS Inline-block

The inline-block value for the display property is a hybrid between inline and block. An element with display: inline-block; behaves like an inline element in that it does not cause a line break, but it also accepts block-level properties such as width and height, making it more versatile than a regular inline element.

Properties of Inline-block:

  • Inline Behavior: An element with inline-block flows inline with surrounding elements, similar to an inline element.
    • span {
    display: inline-block;
}
  • Block Behavior: An element with inline-block can accept width, height, padding, margin, and other block-level properties.
    • div {
    display: inline-block;
    width: 200px;
    height: 100px;
}
  • Aligning Inline-block Elements: You can align inline-block elements horizontally using properties like text-align on their parent element.
    • div {
    text-align: center;
}

span {
    display: inline-block;
}
  • Vertical Alignment: You can control the vertical alignment of inline-block elements using the vertical-align property.
    • span {
    display: inline-block;
    vertical-align: middle;
}

Example of Inline-block Usage:

div {
    text-align: center;
}

span {
    display: inline-block;
    margin: 10px;
    padding: 20px;
    background-color: lightgray;
}

This example centers the inline-block <span> elements within their parent <div>, and each <span> has padding, margin, and background-color applied.

3. CSS Align

The align property in CSS is not a single property, but rather a concept used across different contexts to align elements in various ways. Different properties are used for aligning elements in different scenarios, such as text, blocks, flex items, or grid items. Here, we will explore how alignment works with text-align, vertical-align, flexbox, and grid.

Aligning Text:

  • text-align: The text-align property is used to align text horizontally within its container. It works on block-level elements like <div>, <section>, and inline elements like <span>.
    • p {
    text-align: center;
}
  • Possible Values: The text-align property accepts values such as left, center, right, and justify.
    • p {
    text-align: justify;
}

Aligning Inline or Inline-block Elements:

  • vertical-align: The vertical-align property is used to align inline or inline-block elements vertically relative to their parent or sibling elements. It is commonly used to align images, text, or other inline elements.
    • img {
    vertical-align: middle;
}
  • Possible Values: Common values for vertical-align are top, middle, bottom, and baseline.

Aligning with Flexbox:

  • align-items: The align-items property is used in flexbox layouts to align items along the cross axis (vertically by default). This property affects all flex items within a flex container.
    • div {
    display: flex;
    align-items: center;
}
  • align-self: The align-self property allows you to override the alignment of a specific flex item, independently of the other items.
    • div {
    display: flex;
}

span {
    align-self: flex-start;
}

Aligning with Grid:

  • align-items (Grid): In grid layouts, the align-items property is used to align all grid items along the block (vertical) axis.
    • div {
    display: grid;
    align-items: center;
}
  • align-self (Grid): Like in flexbox, align-self can be used to align individual grid items.
    • span {
    align-self: start;
}
