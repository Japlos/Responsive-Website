1. CSS Display

The display property in CSS is used to define how an element is displayed on the page. It determines the behavior of an element and how it interacts with other elements. The most commonly used display values include block, inline, inline-block, flex, grid, and none.

Common Display Values:

  • block: An element with display: block; will take up the full width of its parent container and create a line break before and after the element.
    • div {
    display: block;
}
  • inline: An element with display: inline; will only take up as much width as its content and will not cause a line break. It flows inline with other elements.
    • span {
    display: inline;
}
  • inline-block: An element with display: inline-block; behaves like an inline element (no line break), but you can set width and height properties.
    • button {
    display: inline-block;
}
  • flex: An element with display: flex; becomes a flex container, allowing its children to be aligned and distributed using the flexbox model.
    • div {
    display: flex;
}
  • grid: An element with display: grid; becomes a grid container, enabling the use of the CSS Grid Layout system to position child elements.
    • section {
    display: grid;
}
  • none: The display: none; value removes the element from the document layout, meaning it doesn't take up any space and is not visible.
    • div {
    display: none;
}

Example of Display Usage:

div {
    display: flex;
    justify-content: center;
    align-items: center;
}

This example sets the <div> to use flexbox, aligning its children in the center horizontally and vertically.

2. CSS Max-width

The max-width property is used to set the maximum width of an element. It ensures that the element does not grow beyond a specified width, regardless of its content or the width of its parent container. This is particularly useful in responsive web design to prevent elements from becoming too wide on larger screens.

Key Points about Max-width:

  • max-width: Defines the maximum width of an element. The element will not exceed this width, even if its content or container would allow it to grow further.
    • img {
    max-width: 100%;
}

    In this example, the image will never exceed the width of its container, making it responsive.

  • Values for max-width can be set in various units such as px, em, %, vw, and others.
    • div {
    max-width: 600px;
}

    This example ensures that the <div> will never exceed 600px in width, even on larger screens.

  • Responsive Design: The max-width property is commonly used in conjunction with width: 100% to ensure elements are flexible and adapt to various screen sizes.
    • img {
    width: 100%;
    max-width: 500px;
}

    This combination ensures that the image is responsive, but it will never be larger than 500px in width, regardless of the screen size.

Example of Max-width Usage:

div {
    max-width: 80%;
    margin: 0 auto;
}

This example sets the <div> to have a maximum width of 80% of its parent container, with automatic margins to center it horizontally.

3. CSS Position

The position property in CSS determines how an element is positioned in the document. By default, elements are positioned statically, but you can change this behavior using different position values like absolute, relative, fixed, sticky, and static.

Positioning Types:

  • static: This is the default positioning for all elements. Elements with position: static; are positioned based on the normal flow of the document and are not affected by top, bottom, left, or right properties.
    • div {
    position: static;
}
  • relative: An element with position: relative; is positioned relative to its normal position in the document flow. This allows you to offset the element from where it would normally appear using the top, right, bottom, and left properties.
    • div {
    position: relative;
    top: 10px;
    left: 20px;
}

    This example moves the <div> element 10px down and 20px to the right relative to its normal position.

  • absolute: An element with position: absolute; is positioned relative to its closest positioned ancestor (an element with a position other than static). If no positioned ancestor is found, it is positioned relative to the initial containing block (usually the <html> element).
    • div {
    position: absolute;
    top: 50px;
    left: 100px;
}

    This example places the <div> 50px from the top and 100px from the left of its positioned ancestor (or the page if no positioned ancestor is found).

  • fixed: An element with position: fixed; is positioned relative to the browser window (viewport), and it stays in the same place even when the page is scrolled.
    • div {
    position: fixed;
    top: 0;
    right: 0;
}

    This example fixes the <div> element at the top-right corner of the page, and it will remain in that position even when the user scrolls.

  • sticky: An element with position: sticky; toggles between relative and fixed, depending on the user's scroll position. It acts as a relative element until a specified scroll position is reached, at which point it becomes fixed.
    • div {
    position: sticky;
    top: 0;
}

    This example makes the <div> element "stick" to the top of the page when the user scrolls past it, but it behaves as a relatively positioned element before the scroll position is reached.

Example of Position Usage:

div {
    position: absolute;
    top: 50px;
    right: 50px;
}

This example positions the <div> element 50px from the top and 50px from the right of its nearest positioned ancestor.
