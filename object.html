1. What is a JavaScript Object?

A JavaScript object is a collection of key-value pairs, where each key (also called a property) is a string and the value can be any valid JavaScript data type, including numbers, strings, arrays, other objects, or functions. Objects are fundamental in JavaScript and are used to store and organize data.

1.1 Object Syntax

Objects are created using curly braces ({}) and consist of a list of key-value pairs separated by commas. Each key is followed by a colon, then the corresponding value.

let person = {
    name: "John",
    age: 30,
    city: "New York"
};

In this example, person is an object with three properties: name, age, and city.

2. JavaScript Object Properties

Properties are values associated with an object, represented as key-value pairs. The key (also known as the property name) is a string, and the value can be any JavaScript data type.

2.1 Accessing Object Properties

Object properties can be accessed in two ways: dot notation or bracket notation.

Dot Notation

console.log(person.name); // Outputs: John

Bracket Notation

console.log(person["age"]); // Outputs: 30

Dot notation is preferred for most cases, but bracket notation is useful when the property name is dynamic or contains special characters (e.g., spaces or hyphens).

2.2 Modifying Object Properties

Properties can be updated by assigning a new value to the existing property using dot or bracket notation.

person.city = "Los Angeles"; // Updates the city property

2.3 Adding New Properties

New properties can be added to an object at any time using dot or bracket notation.

person.country = "USA"; // Adds a new property 'country'

Alternatively, you can use bracket notation for adding properties with special characters in the name:

person["postal-code"] = "90001";

3. JavaScript Object Methods

Methods are functions that are stored as object properties. Methods are used to perform operations on an object’s data or return a value. You can define a method inside an object just like a property, but instead of a value, you assign a function to the key.

3.1 Defining Methods

Methods are defined just like any other property, except that the value is a function. You can define methods using function expressions or shorthand method syntax.

let person = {
    name: "John",
    greet: function() {
        console.log("Hello, " + this.name);
    }
};

person.greet(); // Outputs: Hello, John

3.2 Method Shorthand Syntax

Since ES6 (ECMAScript 2015), JavaScript allows a shorthand syntax to define methods inside objects.

let person = {
    name: "John",
    greet() {
        console.log("Hello, " + this.name);
    }
};

person.greet(); // Outputs: Hello, John

The this keyword inside the method refers to the object the method belongs to.

3.3 Using Methods with Other Objects

Methods can also be used to interact with other objects, including modifying other properties or calling other methods inside the same object.

let person = {
    name: "John",
    age: 30,
    greet() {
        console.log("Hello, " + this.name);
    },
    incrementAge() {
        this.age++;
    }
};

person.greet(); // Outputs: Hello, John
person.incrementAge();
console.log(person.age); // Outputs: 31

4. JavaScript Object Display

To display an object in JavaScript, you can use console.log(), which outputs the object's structure, properties, and values to the console. However, displaying objects directly in the DOM requires converting the object into a string representation.

4.1 Displaying Objects in the Console

let person = {
    name: "John",
    age: 30,
    city: "New York"
};

console.log(person); // Outputs: { name: "John", age: 30, city: "New York" }

4.2 Displaying Objects in the DOM

If you want to display the contents of an object on a web page, you need to convert the object into a readable format. One common method is to use JSON.stringify() to convert the object into a string, which can be displayed in the DOM.

let person = {
    name: "John",
    age: 30,
    city: "New York"
};

document.getElementById("output").innerText = JSON.stringify(person);

This code converts the person object into a string and displays it in an HTML element with the id output.

5. JavaScript Object Constructors

Object constructors are special functions used to create and initialize objects. The constructor function acts as a blueprint for creating objects with the same properties and methods. You can create a new instance of an object by using the new keyword followed by a constructor function.

5.1 Creating Objects with a Constructor Function

In JavaScript, you can create an object using a constructor function. The function defines the properties of the object, and you can instantiate objects from the constructor function.

function Person(name, age) {
    this.name = name;
    this.age = age;
    this.greet = function() {
        console.log("Hello, " + this.name);
    };
}

let person1 = new Person("John", 30);
let person2 = new Person("Alice", 25);

person1.greet(); // Outputs: Hello, John
person2.greet(); // Outputs: Hello, Alice

5.2 Constructor Function vs Object Literal

Both constructor functions and object literals can be used to create objects. However, constructor functions are more useful when you need to create multiple instances of objects with the same properties and methods.

// Object Literal
let person1 = {
    name: "John",
    age: 30,
    greet: function() {
        console.log("Hello, " + this.name);
    }
};

// Constructor Function
function Person(name, age) {
    this.name = name;
    this.age = age;
    this.greet = function() {
        console.log("Hello, " + this.name);
    };
}
let person2 = new Person("Alice", 25);

5.3 Using the class Syntax (ES6)

ES6 introduced the class syntax, which is a more concise way to define constructor functions. It is a syntactic sugar over JavaScript’s prototype-based inheritance and offers a more object-oriented approach to creating objects.

class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }
    greet() {
        console.log("Hello, " + this.name);
    }
}

let person1 = new Person("John", 30);
let person2 = new Person("Alice", 25);

person1.greet(); // Outputs: Hello, John
person2.greet(); // Outputs: Hello, Alice
