1. CSS Overflow

The overflow property in CSS controls what happens when content overflows an element's box. This property is used when the content inside an element is too large to fit into the specified dimensions (width and height). The overflow property can help manage how the extra content is displayed or hidden.

Possible Values for Overflow:

  • visible: This is the default value. The overflowed content is visible outside the element's box.
    • div {
    overflow: visible;
}
  • hidden: The overflowed content is clipped, and no scrollbars are provided to view the hidden content.
    • div {
    overflow: hidden;
}
  • scroll: The overflowed content will be clipped, but scrollbars will appear to allow the user to scroll and view the content.
    • div {
    overflow: scroll;
}
  • auto: The browser will automatically add scrollbars if the content overflows the element's box. This is the most commonly used value for scrollable content.
    • div {
    overflow: auto;
}
  • overflow-x: Specifies the overflow behavior on the horizontal axis (left-to-right).
    • div {
    overflow-x: scroll;
}
  • overflow-y: Specifies the overflow behavior on the vertical axis (top-to-bottom).
    • div {
    overflow-y: hidden;
}

Example of Overflow Usage:

div {
    width: 200px;
    height: 100px;
    overflow: auto;
}

This example creates a <div> with a fixed width and height. If the content inside the <div> exceeds these dimensions, scrollbars will appear to allow the user to scroll and view the overflowed content.

2. CSS Z-index

The z-index property in CSS is used to control the stacking order of elements that overlap. It determines which elements appear in front and which ones are stacked behind others. Elements with a higher z-index value are displayed in front of those with a lower value.

Key Points about Z-index:

  • Default Value: The default value of z-index is auto, meaning the stacking order is based on the order in which elements appear in the HTML.
    • div {
    z-index: auto;
}
  • Positive Values: A positive z-index will place the element above elements with a lower z-index value. The higher the number, the closer the element is to the front.
    • div {
    position: absolute;
    z-index: 10;
}
  • Negative Values: A negative z-index will place the element below elements with a higher z-index value.
    • div {
    position: absolute;
    z-index: -1;
}
  • z-index and Positioning: For z-index to work, the element must have a positioning value other than static (e.g., relative, absolute, or fixed).
    • div {
    position: relative;
    z-index: 5;
}

Example of Z-index Usage:

div {
    position: absolute;
    z-index: 10;
}

p {
    position: absolute;
    z-index: 5;
}

This example positions two elements absolutely on the page. The <div> will appear above the <p> element because it has a higher z-index.

3. CSS Float

The float property in CSS is used to place elements to the left or right of their containing block, allowing other content to wrap around them. Floats are typically used for layouts, like positioning images next to text. However, they can be tricky to use correctly, and modern layout techniques like Flexbox and Grid are often preferred over floats.

Float Values:

  • left: This value positions the element to the left of its containing block, allowing text or other elements to wrap around it on the right.
    • img {
    float: left;
}
  • right: This value positions the element to the right of its containing block, allowing text or other elements to wrap around it on the left.
    • img {
    float: right;
}
  • none: This value cancels any floating behavior, and the element will be positioned according to the normal document flow.
    • img {
    float: none;
}
  • clear: The clear property is used to control the behavior of elements around floating elements. It is commonly used to prevent elements from wrapping around a float.
    • p {
    clear: both;
}

Example of Float Usage:

img {
    float: left;
    margin-right: 20px;
}

p {
    clear: both;
}

This example positions an image on the left of the container, with text wrapping around it on the right. The clear: both ensures that the <p> element is displayed below any floated elements.
