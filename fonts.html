1. CSS Text

The CSS text properties are used to control the style and behavior of text content within an element. These properties can affect how text is displayed, its alignment, decoration, transformation, and spacing.

Key Properties of CSS Text:

  • text-align: Defines the horizontal alignment of text inside an element. Possible values include left, right, center, and justify.
    • text-align: center;
  • text-decoration: Specifies decoration on text, such as underline, overline, line-through, or none.
    • text-decoration: underline;
  • text-transform: Controls the capitalization of text. Possible values are uppercase, lowercase, and capitalize.
    • text-transform: uppercase;
  • letter-spacing: Adjusts the space between characters of the text. You can use it to improve readability or create artistic effects.
    • letter-spacing: 2px;
  • line-height: Specifies the height of a line of text, used for controlling the spacing between lines in multi-line text.
    • line-height: 1.5;
  • text-indent: Defines the indentation of the first line of text inside an element.
    • text-indent: 20px;
  • white-space: Defines how white spaces inside an element are handled. Values include normal, nowrap, pre, etc.
    • white-space: nowrap;
  • word-spacing: Controls the space between words in a text element.
    • word-spacing: 5px;

Example of Text Usage:

p {
    text-align: center;
    text-decoration: underline;
    text-transform: capitalize;
    line-height: 1.8;
}

This example sets the text in a <p> element to be centered, underlined, with capitalized words and increased line height.

2. CSS Fonts

CSS allows you to style text by applying custom fonts. The font properties define the appearance of text within an element, such as the typeface, size, weight, style, and more.

Key Properties of CSS Fonts:

  • font-family: Specifies the font of the text. You can list multiple font names to specify fallback fonts if the first choice is unavailable.
    • font-family: 'Arial', sans-serif;
  • font-size: Defines the size of the font. It can be set using absolute units (px, em) or relative units (rem, %) to make the text responsive.
    • font-size: 16px;
  • font-weight: Specifies the thickness of the font. Common values include normal, bold, and numeric values ranging from 100 to 900.
    • font-weight: bold;
  • font-style: Defines whether the font is italicized or normal.
    • font-style: italic;
  • font-variant: Controls the use of small-caps (uppercase letters in a smaller size than normal). It can be set to normal or small-caps.
    • font-variant: small-caps;
  • line-height: Similar to the text-line-height property, it controls the vertical spacing between lines in a block of text.
    • line-height: 1.6;
  • font-feature-settings: A property that allows advanced typographic features like ligatures and kerning to be specified for supported fonts.
    • font-feature-settings: 'liga' 1;

Example of Font Usage:

h1 {
    font-family: 'Times New Roman', serif;
    font-size: 32px;
    font-weight: bold;
    font-style: italic;
}

This example defines the font for the <h1> element, with "Times New Roman" as the first choice, bold weight, and italic style.

3. CSS Icons

Icons are visual symbols that can be used to enhance the user experience and improve the visual design of a webpage. In CSS, icons are often implemented as images, font-based icons, or SVGs (Scalable Vector Graphics).

Using Font Icons (e.g., Font Awesome):

Font-based icons are popular because they are scalable and easy to style with CSS. One of the most popular libraries is Font Awesome, which provides a variety of vector icons.

Example of Using Font Awesome Icons:

To use Font Awesome, you would need to include the library via CDN or locally in your project. Below is an example of adding an icon to a page:

<i class="fa fa-home"></i>

This example adds a "home" icon from the Font Awesome library using the <i> tag with a class of fa fa-home.

Using SVG Icons:

SVG icons are another method of using scalable graphics. They can be easily customized with CSS. Here's an example of embedding an SVG icon directly into HTML:

<svg xmlns="http://www.w3.org/2000/svg" width="24" height="24" viewBox="0 0 24 24">
    <path d="M12 2C6.48 2 2 6.48 2 12s4.48 10 10 10 10-4.48 10-10S17.52 2 12 2zm0 18c-4.41 0-8-3.59-8-8s3.59-8 8-8 8 3.59 8 8-3.59 8-8 8z"></path>
</svg>

This example shows an SVG icon with a simple circular path and a view box. You can customize the color and size of SVGs using CSS.

4. CSS Links

CSS can be used to style links, including those that are normal, visited, hovered, or active. These different states allow for an interactive, user-friendly experience for navigation elements.

Link States:

  • a: Targets the normal (unvisited) state of the link.
    • a {
    color: blue;
}
  • a:visited: Targets links that have been visited by the user. Usually, this state is styled differently for better UX.
    • a:visited {
    color: purple;
}
  • a:hover: Styles the link when the user hovers over it.
    • a:hover {
    color: red;
}
  • a:active: Defines the style of the link when it is being clicked or active.
    • a:active {
    color: green;
}

Example of Styling Links:

a {
    text-decoration: none;
    color: #3498db;
}

a:hover {
    text-decoration: underline;
    color: #2ecc71;
}

a:visited {
    color: #9b59b6;
}

This example removes the underline from links by default, changes the color to blue, and adds an underline and changes the color when the user hovers over the link.
