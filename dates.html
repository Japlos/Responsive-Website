1. JavaScript Dates

The Date object in JavaScript is used to work with dates and times. It allows you to get the current date and time, manipulate it, and format it in various ways.

1.1. Creating Dates

There are several ways to create a Date object:

  • Current Date and Time: To create a Date object that represents the current date and time, use new Date().
    • const currentDate = new Date();
  • Specific Date: To create a date for a specific time, pass individual components like year, month, day, etc. Remember, months are 0-indexed (January = 0, December = 11).
    • const specificDate = new Date(2024, 10, 5); // November 5, 2024
  • Date from String: You can create a Date object from a date string in various formats, such as ISO 8601.
    • const dateFromString = new Date("2024-11-05T12:00:00Z");
  • Milliseconds since Epoch: To create a date from milliseconds since January 1, 1970 (the Unix epoch), you can pass a number representing the milliseconds.
    • const dateFromMilliseconds = new Date(1668760800000);

1.2. Working with Dates

Once you have a Date object, you can perform operations such as getting the current year, month, day, and more.

  • Get Current Year: Use the getFullYear() method.
    • const year = currentDate.getFullYear();
  • Get Current Month: Use the getMonth() method (returns a zero-indexed month).
    • const month = currentDate.getMonth();
  • Get Current Day: Use the getDate() method to get the day of the month.
    • const day = currentDate.getDate();
  • Get Day of the Week: Use the getDay() method (returns 0 for Sunday, 6 for Saturday).
    • const dayOfWeek = currentDate.getDay();
  • Get Hours, Minutes, Seconds, Milliseconds: Use methods like getHours(), getMinutes(), etc.
    • const hours = currentDate.getHours();
    const minutes = currentDate.getMinutes();
    const seconds = currentDate.getSeconds();

2. JavaScript Date Formats

JavaScript allows various ways to format and display dates. Some common formats include:

  • ISO 8601 Format: This format follows the pattern YYYY-MM-DDTHH:mm:ss.sssZ and is commonly used for machine-readable date strings.
    • const isoString = currentDate.toISOString();
  • Locale-Specific Format: The toLocaleString() method returns a string in the format suitable for the current locale.
    • const localeString = currentDate.toLocaleString();
  • Short Date Format: Use toLocaleDateString() to get the date in a localized short format.
    • const shortDate = currentDate.toLocaleDateString();
  • Short Time Format: Use toLocaleTimeString() to get the time in a localized short format.
    • const shortTime = currentDate.toLocaleTimeString();
  • UTC Format: The toUTCString() method returns the date in a readable UTC string.
    • const utcString = currentDate.toUTCString();
  • Custom Format (via toString()): The toString() method provides a human-readable string of the date.
    • const dateString = currentDate.toString();

3. JavaScript Get and Set Methods

JavaScript provides a variety of get and set methods to retrieve and modify specific parts of a Date object.

3.1. Get Methods

These methods allow you to get individual parts of a date, such as the year, month, day, etc.

  • getFullYear(): Returns the year of the date.
    • const year = currentDate.getFullYear();
  • getMonth(): Returns the month (0-11). January is 0, December is 11.
    • const month = currentDate.getMonth();
  • getDate(): Returns the day of the month (1-31).
    • const day = currentDate.getDate();
  • getDay(): Returns the day of the week (0-6), where 0 is Sunday.
    • const weekday = currentDate.getDay();
  • getHours(), getMinutes(), getSeconds(), getMilliseconds(): These methods return the respective time unit values (in the current timezone).
    • const hours = currentDate.getHours();
  • getTime(): Returns the number of milliseconds since January 1, 1970 (UTC).
    • const time = currentDate.getTime();
  • getTimezoneOffset(): Returns the time zone difference in minutes from UTC.
    • const timezoneOffset = currentDate.getTimezoneOffset();

3.2. Set Methods

These methods allow you to modify specific parts of a date.

  • setFullYear(year, month, day): Sets the year, month, and day.
    • currentDate.setFullYear(2025);
  • setMonth(month, day): Sets the month and optionally the day.
    • currentDate.setMonth(5, 15);
  • setDate(day): Sets the day of the month.
    • currentDate.setDate(10);
  • setHours(hours, minutes, seconds, milliseconds): Sets the hour, minute, second, and millisecond values.
    • currentDate.setHours(9, 30, 0, 0);
  • setMinutes(minutes, seconds, milliseconds): Sets the minutes, seconds, and milliseconds.
    • currentDate.setMinutes(45, 0, 0);
  • setSeconds(seconds, milliseconds): Sets the seconds and milliseconds.
    • currentDate.setSeconds(30, 500);
  • setMilliseconds(milliseconds): Sets the milliseconds.
    • currentDate.setMilliseconds(250);
  • setTime(milliseconds): Sets the date and time using the number of milliseconds since January 1, 1970 (UTC).
    • currentDate.setTime(1668760800000);
