1. JavaScript Operators

In JavaScript, operators are special symbols used to perform operations on values and variables. They are categorized into various types, including arithmetic, assignment, comparison, logical, and more. Below are the main categories of operators:

1.1 Types of Operators

  • Arithmetic Operators: Used to perform mathematical calculations.
  • Assignment Operators: Used to assign values to variables.
  • Comparison Operators: Used to compare two values and return a Boolean value.
  • Logical Operators: Used to combine multiple conditions.
  • Unary Operators: Operate on a single operand.
  • Conditional (Ternary) Operator: A shortcut for an if-else statement.
  • Bitwise Operators: Perform operations on binary representations of integers.
  • Type Operators: Used to determine the type of a variable or expression (e.g., typeof).

2. Arithmetic Operators

Arithmetic operators are used to perform mathematical operations such as addition, subtraction, multiplication, and division. Below are the arithmetic operators available in JavaScript:

2.1 Addition (+)

The + operator adds two operands together.

let sum = 5 + 3; // sum will be 8

2.2 Subtraction (-)

The - operator subtracts the right operand from the left operand.

let difference = 10 - 4; // difference will be 6

2.3 Multiplication (*)

The * operator multiplies two operands.

let product = 6 * 7; // product will be 42

2.4 Division (/)

The / operator divides the left operand by the right operand.

let quotient = 20 / 4; // quotient will be 5

2.5 Modulus (%)

The % operator returns the remainder of dividing the left operand by the right operand.

let remainder = 9 % 4; // remainder will be 1

2.6 Exponentiation (**)

The ** operator raises the left operand to the power of the right operand.

let power = 2 ** 3; // power will be 8 (2 raised to the power of 3)

2.7 Increment (++)

The ++ operator increases the value of a variable by 1.

let num = 5;
num++; // num will be 6 (incremented by 1)

2.8 Decrement (--)

The -- operator decreases the value of a variable by 1.

let num = 5;
num--; // num will be 4 (decremented by 1)

3. Assignment Operators

Assignment operators are used to assign values to variables. In JavaScript, the most commonly used assignment operator is the simple =, but there are several other assignment operators that allow for shorthand operations.

3.1 Simple Assignment (=)

The = operator is used to assign a value to a variable.

let x = 10; // Assigns the value 10 to the variable 'x'

3.2 Addition Assignment (+=)

The += operator adds the right operand to the left operand and assigns the result to the left operand.

let x = 5;
x += 3; // x will be 8 (5 + 3)

3.3 Subtraction Assignment (-=)

The -= operator subtracts the right operand from the left operand and assigns the result to the left operand.

let x = 10;
x -= 4; // x will be 6 (10 - 4)

3.4 Multiplication Assignment (*=)

The *= operator multiplies the left operand by the right operand and assigns the result to the left operand.

let x = 5;
x *= 2; // x will be 10 (5 * 2)

3.5 Division Assignment (/=)

The /= operator divides the left operand by the right operand and assigns the result to the left operand.

let x = 10;
x /= 2; // x will be 5 (10 / 2)

3.6 Modulus Assignment (%=)

The %= operator calculates the remainder of dividing the left operand by the right operand and assigns the result to the left operand.

let x = 10;
x %= 3; // x will be 1 (10 % 3)

3.7 Exponentiation Assignment (**=)

The **= operator raises the left operand to the power of the right operand and assigns the result to the left operand.

let x = 2;
x **= 3; // x will be 8 (2 raised to the power of 3)

4. JavaScript Operator Precedence

Operator precedence determines the order in which operators are evaluated in an expression. Arithmetic operators have different levels of precedence, with some being evaluated before others. For example, multiplication and division have a higher precedence than addition and subtraction.

4.1 Example of Operator Precedence:

let result = 5 + 3 * 2; 
// result will be 11 because multiplication has higher precedence than addition

4.2 Parentheses

You can override operator precedence by using parentheses, ensuring that certain operations are performed first.

let result = (5 + 3) * 2; 
// result will be 16 because the addition inside the parentheses is evaluated first
