HTML Overview

HTML (HyperText Markup Language) is the standard markup language used to create web pages. This section covers the foundational concepts of HTML that every beginner should know.

Essential HTML Tags

These tags are the core building blocks of HTML:

  • <html>: The root element that defines the document as HTML.
  • <head>: Contains meta-information, links to stylesheets, etc.
  • <title>: Specifies the page title shown in the browser tab.
  • <body>: Holds all visible content on the webpage.

Text Formatting Tags

Common tags used for text formatting include:

  • <h1> to <h6>: Headings, with <h1> as the largest.
  • <p>: Paragraphs, used for regular text blocks.
  • <strong>: Bold text to emphasize importance.
  • <em>: Italic text for emphasis.

HTML Images

The <img> tag is used to embed images in an HTML document. It is a self-closing tag that requires at least the src and alt attributes.

  • src: Specifies the path to the image file.
  • alt: Provides a text alternative if the image cannot be displayed. This is important for accessibility.

Example:

          <img src="images/html_logo.png" alt="HTML Logo" width="200" height="200">

The above example would display an image of the HTML logo if the path is correct.

HTML Links

The <a> (anchor) tag creates a hyperlink to other pages, websites, or sections within the same page. The href attribute specifies the destination URL or path.

  • href: Defines the link's destination. This can be an external URL or an internal page.
  • target: Specifies where to open the link. _blank opens it in a new tab.

Example:

          <a href="https://www.example.com" target="_blank">Visit Example Site</a>

This link will open "https://www.example.com" in a new tab.

Understanding HTML Attributes

Attributes provide additional information about elements. The most common attributes include:

  • id: Uniquely identifies an element on a page.
  • class: Groups elements with similar styling.
  • src: Specifies the source file for images, audio, or video.
  • href: Links to another page or section.

HTML Structure Example

    <!DOCTYPE html>
    <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Sample Page</title>
        </head>
        <body>
            <h1>Hello World</h1>
            <p>This is a paragraph in HTML.</p>
        </body>
    </html>
