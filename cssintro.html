1. What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to control the presentation, layout, and design of HTML documents. CSS defines how HTML elements should be displayed on screen, paper, or in other media. It is responsible for aspects like colors, fonts, spacing, alignment, and responsiveness of the web page.

Purpose of CSS:

  • Separation of content and presentation: HTML provides the structure of the document, while CSS is used for styling.
  • Consistency: CSS allows for consistent styling across multiple pages of a website.
  • Improves maintainability: With CSS, changes can be made in one place (the stylesheet) rather than editing individual HTML elements.

2. CSS Syntax

CSS follows a specific syntax to apply styles to HTML elements. The basic structure of a CSS rule consists of a selector and a declaration block:

Basic Syntax:

selector {
    property: value;
}

The selector is used to target the HTML element, and the declaration block contains the property and its corresponding value.

Example:

p {
    color: red;
    font-size: 16px;
}

This CSS rule selects all <p> elements and sets their text color to red and font size to 16 pixels.

3. Types of CSS

CSS can be applied to an HTML document in three different ways: Inline CSS, Internal CSS, and External CSS.

Inline CSS:

Inline CSS is applied directly within an HTML element using the style attribute. 

<p style="color: blue; font-size: 14px;">This is a paragraph with inline CSS.</p>

Internal CSS:

Internal CSS is placed inside a <style> element within the <head> section of an HTML document. 

<style>
    p {
        color: green;
        font-size: 18px;
    }
</style>

External CSS:

External CSS is written in a separate .css file and linked to the HTML document using the <link> element. 

<link rel="stylesheet" href="styles.css">

External CSS is the preferred method for larger projects because it allows for easier maintenance and faster page loading (due to caching).

4. CSS Selectors

CSS selectors are used to target specific HTML elements to apply styles. Here are some common types of selectors:

Universal Selector:

The universal selector * selects all elements on a page. 

* {
    color: black;
}

Type Selector (Element Selector):

The type selector targets elements based on their tag name.

h1 {
    font-size: 24px;
}

Class Selector:

The class selector selects elements with a specific class attribute. Class selectors are prefixed with a period (.) and can be applied to multiple elements. 

.class-name {
    color: blue;
}

ID Selector:

The ID selector selects elements with a specific id attribute. ID selectors are prefixed with a hash (#) and should be unique within a page. 

#id-name {
    font-size: 20px;
}

Attribute Selector:

Attribute selectors target elements based on their attributes.

a[href^="https"] {
    color: green;
}

Descendant Selector:

A descendant selector targets elements that are inside (nested within) a specified parent element. 

div p {
    color: red;
}

5. Common CSS Properties

CSS properties define the style and appearance of HTML elements. Here are some common properties:

Color and Background:

  • color: Sets the text color.
  • background-color: Sets the background color of an element.
  • background-image: Sets an image as the background of an element.
  • opacity: Defines the transparency level of an element.

Text and Font:

  • font-family: Specifies the font type.
  • font-size: Defines the size of the text.
  • font-weight: Sets the weight (boldness) of the text.
  • line-height: Specifies the height of each line of text.
  • text-align: Aligns the text (left, right, center, justify).

Spacing and Box Model:

  • margin: Creates space outside the border of an element.
  • padding: Creates space inside the border of an element.
  • border: Defines the border around an element.
  • width: Specifies the width of an element.
  • height: Specifies the height of an element.

Positioning:

  • position: Specifies how an element is positioned (static, relative, absolute, fixed, sticky).
  • top, right, bottom, left: Define the position of an element when position is set to something other than static.

Display and Visibility:

  • display: Defines how an element is displayed (block, inline, inline-block, none).
  • visibility: Specifies whether an element is visible or hidden, but it still takes up space on the page.

6. CSS Box Model

The CSS Box Model defines the rectangular boxes generated for elements and consists of the following parts:

  • Content: The actual content of the element (text, images, etc.).
  • Padding: The space between the content and the border.
  • Border: The border surrounding the padding (if defined).
  • Margin: The space outside the border, separating the element from other elements.

It's important to understand the box model because it affects how elements are sized and how they interact with surrounding elements.

7. CSS Media Queries

Media queries are a powerful feature in CSS that allows you to apply different styles based on conditions like screen size, device type, or orientation. Media queries enable responsive web design by making your website adapt to different screen sizes.

Basic Syntax of Media Query:

@media (max-width: 600px) {
    body {
        font-size: 14px;
    }
}

This example applies a font size of 14px to the body element when the screen width is less than or equal to 600px.
