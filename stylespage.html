Introduction to HTML Styles

Styling HTML allows you to change the appearance of your webpage. HTML supports several methods for styling content, including inline styles, internal CSS, and external CSS. Each method has its own uses and benefits.

Inline Styles

Inline styles apply directly within an HTML element using the style attribute. This method is useful for quick, one-off styling.

Example:

        <p style="color: blue; font-size: 20px;">This is styled text.</p>

Internal CSS

Internal CSS is defined within the <style> tag inside the <head> section. It is useful for styling a single HTML document with multiple rules.

Example:

        <style>
          p {
            color: green;
            font-size: 18px;
          }
        </style>

External CSS

External CSS uses a separate stylesheet file linked to the HTML document, allowing consistent styling across multiple pages. It is the recommended method for styling a large website.

Example link to CSS:

        <link rel="stylesheet" href="style.css">

Common CSS Properties

Text Styling

  • color: Sets the color of text. color: blue;
  • font-size: Defines the size of text. font-size: 20px;
  • font-weight: Makes text bold. font-weight: bold;

Background Styling

  • background-color: Sets the background color. background-color: lightgray;
  • background-image: Adds a background image. background-image: url('image.jpg');

Box Model Properties

  • padding: Space inside the element border. padding: 10px;
  • margin: Space outside the element border. margin: 20px;
  • border: Defines the border. border: 1px solid black;
