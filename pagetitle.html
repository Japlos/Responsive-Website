1. HTML Page Titles

The <title> tag is used to define the title of an HTML page. The title appears in the browser's title bar or tab and is important for SEO (Search Engine Optimization).

Usage:

<head>
    <title>Your Page Title Here</title>
</head>

Important Points:

  • Placement: The <title> tag must be placed within the <head> section of the HTML document.
  • Length: It is recommended to keep the title between 50-60 characters, as most search engines truncate titles longer than that.
  • Relevance: The title should be relevant to the content of the page and reflect its main topic.
  • SEO: Include relevant keywords in the title for better search engine ranking.
  • Uniqueness: Each page should have a unique title to help differentiate it in search engine results.

Example:

<head>
    <title>How to Create a Responsive Web Design</title>
</head>

2. HTML Tables

HTML tables are used for displaying data in a tabular format, consisting of rows and columns. Tables are created using the <table> element, and each row is represented by the <tr> tag. Each column of data is placed within a <td> (table data) tag, while table headings are represented by <th>.

Basic Structure:

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Row 1, Column 1</td>
        <td>Row 1, Column 2</td>
    </tr>
</table>

Table Tags:

  • <table> - Defines the table.
  • <tr> - Defines a table row.
  • <th> - Defines a table header cell.
  • <td> - Defines a table data (cell).

Example:

Product Price Quantity
Apple $1.00 10
Banana $0.50 20
Cherry $2.00 15

Additional Table Elements:

  • <thead> - Groups the header content in a table.
  • <tbody> - Groups the body content in a table.
  • <tfoot> - Groups the footer content in a table.
  • <caption> - Adds a title to the table.

3. HTML Lists

HTML lists are used to group items in an ordered (numbered) or unordered (bulleted) manner. There are three types of lists: ordered lists, unordered lists, and definition lists.

Unordered List:

An unordered list is used for items that do not have a specific order. It is defined using the <ul> tag, and each item in the list is placed inside an <li> (list item) tag.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

Ordered List:

An ordered list is used when the order of the list items matters. It is created using the <ol> tag, and each list item is placed inside an <li> tag.

<ol>
    <li>First step</li>
    <li>Second step</li>
    <li>Third step</li>
</ol>

Definition List:

A definition list is used to describe terms and their definitions. It is created using the <dl> tag. Each term is placed in the <dt> tag, and the definition is placed in the <dd> tag.

<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
</dl>

List Styles:

  • <ul> - Defines an unordered (bulleted) list.
  • <ol> - Defines an ordered (numbered) list.
  • <li> - Defines a list item (used in both ordered and unordered lists).
  • <dl> - Defines a definition list.
  • <dt> - Defines a term in a definition list.
  • <dd> - Defines a description or definition for a term in a definition list.

Example:

Unordered List:

  • Apples
  • Bananas
  • Cherries

Ordered List:

  1. First item
  2. Second item
  3. Third item

Definition List:

HTML
HyperText Markup Language
CSS
Cascading Style Sheets
