1. What is a JavaScript String?

A string in JavaScript is a sequence of characters enclosed within either single quotes (' '), double quotes (" "), or backticks (` `). Strings are used to represent text in JavaScript.

1.1 String Declaration

Strings can be declared using single, double, or backticks for template literals.

let str1 = "Hello, World!";
let str2 = 'Hello, World!';
let str3 = `Hello, World!`; // Template literals

1.2 String Length

The length of a string can be determined using the length property.

let str = "Hello, World!";
console.log(str.length); // Outputs: 13

2. JavaScript String Methods

JavaScript provides a variety of methods that can be used to manipulate strings. Here are some commonly used string methods:

2.1 toUpperCase() and toLowerCase()

These methods convert the string to uppercase or lowercase.

let str = "hello";
console.log(str.toUpperCase()); // Outputs: HELLO
console.log(str.toLowerCase()); // Outputs: hello

2.2 indexOf()

Returns the index of the first occurrence of a specified substring. Returns -1 if the substring is not found.

let str = "Hello, World!";
console.log(str.indexOf("World")); // Outputs: 7
console.log(str.indexOf("JavaScript")); // Outputs: -1

2.3 slice()

Extracts a section of a string and returns it as a new string.

let str = "Hello, World!";
console.log(str.slice(0, 5)); // Outputs: Hello

2.4 replace()

Replaces a specified value with another value in a string. It only replaces the first occurrence unless a global regular expression is used.

let str = "Hello, World!";
console.log(str.replace("World", "JavaScript")); // Outputs: Hello, JavaScript!

2.5 trim()

Removes whitespace from both ends of a string.

let str = "   Hello, World!   ";
console.log(str.trim()); // Outputs: "Hello, World!"

2.6 concat()

Combines two or more strings and returns a new string.

let str1 = "Hello";
let str2 = "World!";
console.log(str1.concat(", ", str2)); // Outputs: Hello, World!

3. JavaScript Template Literals

Template literals, introduced in ES6, are string literals that allow embedded expressions and multi-line strings. They are defined using backticks (` `) instead of quotes.

3.1 Basic Syntax

let str = `Hello, World!`;
console.log(str); // Outputs: Hello, World!

3.2 String Interpolation

Template literals allow string interpolation, where you can embed variables or expressions inside a string by using ${}.

let name = "Alice";
let greeting = `Hello, ${name}!`;
console.log(greeting); // Outputs: Hello, Alice!

3.3 Multi-line Strings

Template literals can also be used for creating multi-line strings without using escape characters.

let str = `Hello,
World!`;
console.log(str); // Outputs:
                  // Hello,
                  // World!

4. JavaScript Search Methods

JavaScript provides various methods to search within strings. The most commonly used methods are:

4.1 indexOf()

As mentioned before, indexOf() returns the index of the first occurrence of a specified substring, or -1 if the substring is not found.

let str = "Hello, World!";
console.log(str.indexOf("World")); // Outputs: 7

4.2 includes()

The includes() method checks if a specified string is present within another string and returns true or false.

let str = "Hello, World!";
console.log(str.includes("World")); // Outputs: true
console.log(str.includes("JavaScript")); // Outputs: false

4.3 search()

The search() method searches for a match between a regular expression and a string. It returns the index of the match or -1 if no match is found.

let str = "Hello, World!";
console.log(str.search("World")); // Outputs: 7

5. JavaScript Numbers

In JavaScript, numbers are represented as 64-bit floating-point values. JavaScript supports both integer and floating-point numbers, as well as a variety of methods for performing operations on numbers.

5.1 Basic Arithmetic Operations

JavaScript supports basic arithmetic operations such as addition, subtraction, multiplication, division, and modulus (remainder).

let x = 10;
let y = 5;
console.log(x + y); // Outputs: 15
console.log(x - y); // Outputs: 5
console.log(x * y); // Outputs: 50
console.log(x / y); // Outputs: 2
console.log(x % y); // Outputs: 0

5.2 NaN (Not a Number)

In JavaScript, NaN is a special value that represents "Not a Number". It results from invalid number operations such as dividing zero by zero.

console.log(0 / 0); // Outputs: NaN

5.3 parseInt() and parseFloat()

These methods are used to convert a string into a number. parseInt() converts a string to an integer, and parseFloat() converts a string to a floating-point number.

let str1 = "42";
let str2 = "42.42";

console.log(parseInt(str1)); // Outputs: 42
console.log(parseFloat(str2)); // Outputs: 42.42

5.4 isNaN()

The isNaN() function checks if a value is NaN. It returns true if the value is NaN, and false otherwise.

console.log(isNaN(10)); // Outputs: false
console.log(isNaN("Hello")); // Outputs: true

5.5 Number Methods

There are several built-in methods for working with numbers:

5.5.1 toFixed()

Rounds a number to a specified number of decimal places and returns it as a string.

let num = 123.456;
console.log(num.toFixed(2)); // Outputs: 123.46

5.5.2 toPrecision()

Formats a number to a specified length (total number of significant digits).

let num = 123.456;
console.log(num.toPrecision(4)); // Outputs: 123.5

5.5.3 toString()

Converts a number to a string.

let num = 123;
console.log(num.toString()); // Outputs: "123"
